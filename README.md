# TrustiQ Legal Platform

TrustiQ is a bilingual legal platform for GCC jurisdictions. It includes:

- case management
- document drafting and analysis
- AI legal research and chat
- branding and document assets
- Arabic / English UI support

## Run locally

```bash
npm install
npm run build
npm start
```

## Environment

Set at least:

- `OPENAI_API_KEY`
- `JWT_SECRET`

See `.env.example` for more.
